## コードの解説

超絶技巧なプログラムは様々な手法で実現できますが、今回は `eval(%w(...))*""` という基本パターンを使用しています。

この第一歩には、[あなたの知らない超絶技巧プログラミングの世界](https://gihyo.jp/book/2015/978-4-7741-7643-7) という書籍で入門しました。まずは、この本でも紹介されている簡単な例から説明します。

### 基本パターン：eval(%w(...)*"")

最もシンプルな例は以下のコードです：

```ruby
eval(%w(pu
ts      "H
el      lo
,w      or
ld!")*"")#
```


![easy-sample](https://storage.googleapis.com/zenn-user-upload/af5929caf0af-20251215.png)

このプログラムは、`puts "Hello, World!"` を `eval(%w(` と `)*"")` で囲ったものです。
`%w(...)` は単語の配列を作成し、`*""` で連結することで、元のコード `puts "Hello, World!"` として実行されます。先ほど紹介したコードも同じフォーマットで、先頭は `eval(%w(` で始まり、末尾は `)*"")` で終わっています。


### 実際のコードのeval内のプログラムの解説

基本パターンを紹介したので、私が作ったコードについての紹介に戻ります。

コードは以下の3つの部分で構成されています：

1. **文字コード定義**
   ```ruby
   c=32.chr;  # スペース文字を変数cに格納
   a=10.chr; # 改行文字(\n)を変数aに格納
   ```

2. **アスキーアートを圧縮したデータの作成**
   ```ruby
   r="1n18s2h3s2h2s7h2s..."
   ```

3. **デコード＆出力処理**
   ```ruby
   r.scan(/(\d+)([shn])/){art<<($2=="s"?c:$2=="h"?"#":10.chr)*$1.to_i}
   ```

#### 1. ASCIIコードと文字の変換

`32.chr`や `10.chr` は ASCIIコードを文字に変換します：
参考: [instance method Integer#chr](https://docs.ruby-lang.org/ja/latest/method/Integer/i/chr.html)

```ruby
irb(main):001> 32.chr
=> " "
irb(main):002> 10.chr
=> "\n"
irb(main):003> " ".ord
=> 32
irb(main):004> "\n".ord
=> 10
```

#### 2. アスキーアートを圧縮したデータの作成
`r=""` に格納されている `1n18s2h` のような文字列は、以下のような意味を圧縮したデータです：

- `1n` → `\n`という改行1つ
- `18s` → ` `スペース18個
- `2h` → `#`ハッシュ記号2個

この形式でアスキーアートを圧縮して保存しています。

#### 3. デコード＆出力処理

続いてrをデコードして分解していきます。
```ruby
r.scan(/(\d+)([shn])/){art<<($2=="s"?c:$2=="h"?"#":10.chr;)*$1.to_i};
``` 

上記の行で`r=1n18s2h` のような文字列を分解して、`\n   ###` のように文字列に復元させていきます。`shn`のいずれかがあったらそこで正規表現で分解して、sだったら` `を表示 といった条件分岐を書いています。

```ruby
if $2 == "s"
  32.chr
elsif $2 == "h"
  "#"
else
  10.chr
end
```

それを `*$1.to_i`の数だけ掛け算して繰り返し表示しています。

## このロジックを簡単にやるためのテンプレート

これを全部自力で書いていくのは大変なので、少しテンプレートを用意していました。

### アスキーアートの形にプログラムを変更する

以下のコードは、`template` に書いたアスキーアートの形に沿って、任意のコードを流し込むジェネレーターです：

```ruby
# 1. 土台となるアスキーアート（型）
# これで "eval(%w" までが1行に収まり、改行エラーを防ぎます。
template = <<ASCII
            #######
           #######
          #######
         #######
        #######
       #######   #######      ##########   #######   ###    ##    ###    ##
      #######   #######           ##      ##        ####   ##    ####   ##
     #######   #######          ##       #######   ## ##  ##    ## ##  ##
    #######   #######         ##        ##        ##  ## ##    ##  ## ##
   #######   #######        ##         ##        ##   ####    ##   ####
  #######   #######       ##########  #######   ##     ###   ##    ###
 #######   #######
#######   #######
ASCII

# 2. 実行させたいコード
payload = %q{  }
# 3. 実行可能な形式に変換
src = 'eval(%w(' + payload + ')*"")'

# 4. 穴の数チェックとパディング
slots = template.count("#")
diff = slots - src.length

if diff < 0
  puts "エラー: アスキーアートが小さすぎます。#{diff} 個足りません。"
  exit
end

# 足りない分を #xxxx... で埋める
src += "#" * (diff - 1)

# 5. 流し込み
output = template.gsub("#") { src.slice!(0, 1) }

# 6. 出力
puts output
```

このスクリプトは以下の処理を行っています：

1. アスキーアートテンプレート内の `#` の数をカウント
2. 実行したいコードを `eval(%w(...)*"")` で包む
3. コードの長さが足りない場合は `#` でパディング
4. テンプレートの `#` を1文字ずつコードで置き換え

### ステップ2: アスキーアートを圧縮したデータの作成

次に、`Merry Christmas 12.25` のようなアスキーアートの出力を `1n18s2h` 形式へ変換するコードを用意します：

```ruby
template = <<'A'
                  ##   ##  #######  ######   ######   ##   ##
                  ### ###  ##       ##   ##  ##   ##   ## ##
                  #######  #####    ######   ######     ###
                  ## # ##  ##       ##  ##   ##  ##     ###
                  ##   ##  #######  ##   ##  ##   ##    ###

 ######  ##   ##  ######   ######  #######  #######  ##   ##    ###     #####
##       ##   ##  ##   ##    ##    ##         ##     ### ###   ## ##   ##    
##       #######  ######     ##    #######    ##     #######  #######   #####
##       ##   ##  ##  ##     ##         ##    ##     ## # ##  ##   ##       ##
 ######  ##   ##  ##   ##  ######  #######    ##     ##   ##  ##   ##   #####

                   ##    #######         #######   #######
                  ###   ##   ##         ##   ##   ##
                   ##       ##              ##    ######
                   ##     ##             ##           ##
                  ###  #######   ###  #######  ######
A

r = ""
template.each_char.slice_when { |a, b| a != b }.each do |group|
  char = group.first
  key = case char
        when " " then "s"
        when "#" then "h"
        when "\n" then "n"
        end
  r << "#{group.length}#{key}" if key
end

puts r
```

このスクリプトは以下の処理を行っています：

1. アスキーアートを1文字ずつ読み込む
2. 連続する同じ文字をグループ化（`slice_when`）
3. 各グループを「個数+種類」の形式に変換
   - スペース → `s`
   - ハッシュ → `h`
   - 改行 → `n`
4. 圧縮形式の文字列を出力

これで、任意のアスキーアートを圧縮形式に変換できます。
